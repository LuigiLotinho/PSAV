# Hinweise für KI-Entwicklung

Diese Ordnerstruktur ist dauerhafte Anleitung für KI-gestützte Entwicklung.

**Pflicht:** Vor jeder Code-Generierung diesen Ordner vollständig lesen.

**Ziel:** Vibe Coding, Konsistenz vor Cleverness, server-first, HTMX-getrieben, Bootstrap-only, low-JS.

Regeln sind verbindlich. Im Zweifel: einfache, langweilige Lösungen.
